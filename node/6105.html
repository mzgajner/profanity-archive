13. 3. 2014  Reportaža: Bojevniški pust  (1. 3. 2014, Gala Hala Metelkova Ljubljana)
Na pustno soboto, 1. 3. 2014, so v Gala Hali na Metelkovi nastopili finski folk metalci Turisas z ameriško melodic death metal predskupino Starkill, v okviru evropske turneje We Ride Together.

Reportažo je podala Eva Sešek.

Med množico pred vrati Gale Hale je bilo kar nekaj fanov, ki so pokazali svojo podporo s tem, da so si namazali obraze z ''bojno'' barvo (značilna črna/rdeča) Turisasov.

Ko so se vrata ob osmi uri končno odprla, je začela množica počasi kapljati v dvorano. Čakali smo do devetih, da so pripravili še vse potrebno za koncert, nato se je dvorana ovila v dim in na oder so prišli Starkill. Na nastopu predskupine je bilo kar lepo število ljudi, vendar dvorana ni bila popolnoma zapolnjena, kljub temu pa so se Starkill potrudili za dobro vzdušje in pripravili množico, da se je dobro odzvala na njihove zahteve. Skupina je dokaj mlada, odigrali so precej pesmi iz albuma Fires of Life in nam predstavili nov komad iz prihajajočega albuma. Tehnično gledano je bil koncert izpeljan dobro, mogoče na trenutke za nekatere poslušalce preveč monoton, drugače pa fantje pridno pridobivajo na kilometrini.

Ko so zaključili, je bila množica že dobro ogreta, nekaj ljudi pa je kupilo njihove stvari na merch standu. Sledil je približno trideset minutni odmor, da so oder pripravili na Turisas. Vzdušje se je med tem časom močno spremenilo. V dvorani se je zbralo opazno več ljudi, ki so prišli poslušat glavno skupino. Med čakanjem se je velikokrat slišalo vzklikanje imen skupine in žvižganje. Po nestrpnem klicanju publike so se luči ugasnile in oder so zavzeli Turisas.

Mathias "Warlord" Nygård, glavni pevec, je publiko pozdravil in pohvalil fane, ki so se v dvorani zbrali v tako velikem številu. Zaigrali so dvanajst pesmi iz vseh štirih albumov, ki so jih izdali skozi leta. Začeli so s pesmijo 10 More Miles iz njihovega najnovejšega albuma Turisas2013. Med nastopom so imeli dobro spremljavo luči in dimnih učinkov, ki so se skladali z melodijo. Pevec in ostali člani so se aktivno premikali po odru in se velikokrat povezali s publiko. Bolj ko se je bližal konec, bolj je postajalo v Gali Hali vroče. "Warlord"se je z odra vrgel na publiko, nagnil se je h oboževalkam, da so ga te lahko pobožale po glavi in ostalim srečnežem, ki so bili v prvi vrsti, podal roko. Med premorom je povedal svoje mišljenje glede našega piva in pohvalil Slovenijo, kar pa ni bilo slišati kot klišejsko nakladanje.

Svoj nastop so končali s Stand Up and Fight iz njihovega tretjega albuma in cover komadom Rasputin, pri katerem se je publika dokončno razživela in pokazala, kaj res zmore. Ker je po koncertu sledil drug dogodek, se je Gala Hala po zadnjem komadu izpraznila v dobrih 20 minutah. Fani so bili več kot očitno zadovoljni s koncertom, nekateri med njimi so nato še ostali in se še naprej zabavali do zgodnjih jutranjih ur.
